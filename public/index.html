ðŸš€ Quick Start

Configure in your Symfony application:

# config/packages/lemric_plugin.yaml
lemric_plugin:
  marketplace:
    url: 'https://marketplace.lemric.com/'
    cache_ttl: 3600

Install plugins:

# Browse marketplace
bin/console lemric:plugin:browse

# Install plugin
bin/console lemric:plugin:install plugin-name --marketplace --activate

# Check for updates
bin/console lemric:plugin:check-updates
